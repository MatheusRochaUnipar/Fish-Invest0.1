# Fish-Invest

O sistema Fish Invest é uma plataforma de gestão de investimentos pessoais desenvolvida para facilitar o controle financeiro e o acompanhamento de ativos. O objetivo principal do sistema é centralizar informações sobre aportes, rendimentos e diversificação de carteira em uma interface intuitiva e moderna. Através do sistema, o investidor pode realizar seu cadastro, autenticar-se com segurança, gerenciar seus ativos e visualizar o desempenho de suas aplicações em tempo real, auxiliando na tomada de decisão estratégica para o crescimento de seu patrimônio.
